# Olá venda

Projeto desenvolvido durante a aula **Tópicos Especiais em Desenvolvimento de Sistemas II** do curso de **ADS** na **UPF**.

## O que é?

O Olá venda é um projeto de um sistema de vendas, onde o usuário pode cadastrar produtos, clientes e realizar vendas.

## Objetivo

O objetivo do projeto é desenvolver um sistema de vendas, utilizando os conhecimentos adquiridos durante o curso.

## Tecnologias

- [React](https://pt-br.reactjs.org/) - Biblioteca JavaScript para criar interfaces de usuário
- [TypeScript](https://www.typescriptlang.org/) - Superset JavaScript que adiciona tipagem estática
- [React Router](https://reactrouter.com/) - Biblioteca para gerenciar rotas no React
- [React Icons](https://react-icons.github.io/react-icons/) - Biblioteca de ícones para React
- [Emotion](https://emotion.sh/) - Biblioteca para estilização de componentes
- [Axios](https://axios-http.com/ptbr/docs/intro) - Biblioteca para realizar requisições HTTP

## Como executar

### Pré-requisitos

- [Node.js](https://nodejs.org/en/)

### Instalação

```bash
# Clone o repositório
$ git clone

# Entre na pasta do projeto
$ cd olavenda

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm start
```
